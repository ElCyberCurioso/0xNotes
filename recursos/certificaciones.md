# Certificaciones en Ciberseguridad

## Entry Level

### CompTIA Security+
- **Nivel:** Principiante
- **Costo:** ~$400
- **Validez:** 3 años
- **Descripción:** Fundamentos de seguridad IT
- **URL:** [CompTIA Security+](https://www.comptia.org/certifications/security)

### eJPT (eLearnSecurity Junior Penetration Tester)
- **Nivel:** Principiante
- **Costo:** ~$200
- **Validez:** 3 años
- **Descripción:** Introducción práctica al pentesting
- **URL:** [eJPT](https://elearnsecurity.com/product/ejpt-certification/)

## Intermediate

### CEH (Certified Ethical Hacker)
- **Nivel:** Intermedio
- **Costo:** ~$1,199
- **Validez:** 3 años
- **Descripción:** Conocimientos de hacking ético
- **URL:** [CEH](https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/)

### OSCP (Offensive Security Certified Professional)
- **Nivel:** Intermedio-Avanzado
- **Costo:** ~$1,649
- **Validez:** Permanente
- **Descripción:** Pentesting hands-on, 24 horas de examen práctico
- **URL:** [OSCP](https://www.offensive-security.com/pwk-oscp/)
- **Nota:** Muy respetada en la industria

### PNPT (Practical Network Penetration Tester)
- **Nivel:** Intermedio
- **Costo:** ~$399
- **Validez:** Permanente
- **Descripción:** Pentesting práctico con énfasis en reporting
- **URL:** [PNPT](https://certifications.tcm-sec.com/pnpt/)

## Advanced

### OSEP (Offensive Security Experienced Penetration Tester)
- **Nivel:** Avanzado
- **Costo:** ~$1,649
- **Validez:** Permanente
- **Descripción:** Evasión de AV/EDR, técnicas avanzadas
- **URL:** [OSEP](https://www.offensive-security.com/pen300-osep/)

### OSWE (Offensive Security Web Expert)
- **Nivel:** Avanzado
- **Costo:** ~$1,649
- **Validez:** Permanente
- **Descripción:** Seguridad de aplicaciones web avanzada
- **URL:** [OSWE](https://www.offensive-security.com/awae-oswe/)

### OSCE (Offensive Security Certified Expert)
- **Nivel:** Experto
- **Costo:** ~$1,649
- **Validez:** Permanente
- **Descripción:** Exploit development, técnicas avanzadas
- **URL:** [OSCE](https://www.offensive-security.com/exp301-osed/)

## Red Team

### CRTO (Certified Red Team Operator)
- **Proveedor:** Zero-Point Security
- **Costo:** ~$400
- **Descripción:** Red team operations con Cobalt Strike

### CRTP (Certified Red Team Professional)
- **Proveedor:** PentesterAcademy
- **Costo:** ~$249
- **Descripción:** Active Directory pentesting

## Blue Team / Defense

### BTL1 (Blue Team Level 1)
- **Costo:** ~$399
- **Descripción:** Fundamentos de defensa y SOC

### GCIH (GIAC Certified Incident Handler)
- **Proveedor:** SANS
- **Costo:** ~$2,000+
- **Descripción:** Incident response

### GCIA (GIAC Certified Intrusion Analyst)
- **Proveedor:** SANS
- **Costo:** ~$2,000+
- **Descripción:** Network traffic analysis

## Recomendaciones

### Roadmap Sugerido

#### Para Pentesting
1. eJPT (Principiante)
2. PNPT o OSCP (Intermedio)
3. OSEP o OSWE (Avanzado)

#### Para Bug Bounty
1. eJPT
2. Práctica en HackTheBox/TryHackMe
3. OSWE (Web focus)
4. Experiencia práctica en plataformas bug bounty

#### Para Red Team
1. OSCP
2. CRTP (AD focus)
3. CRTO (C2 framework)
4. OSEP (Evasion)

## Tips para Certificaciones

> [!TIP]
> **Preparación:**
> - Practica en labs hands-on
> - Lee writeups y blogs
> - Toma notas organizadas (como estas!)
> - Únete a comunidades (Discord, Reddit)
> - Práctica, práctica, práctica

> [!WARNING]
> **Consideraciones:**
> - Algunas certificaciones son muy caras
> - No todas las certificaciones son igual de valoradas
> - OSCP es la más reconocida pero también difícil
> - Experiencia práctica > certificaciones
